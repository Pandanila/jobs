# JOBS — Промпт для вайбкода

Используй для: **Bolt.new / Lovable / v0 / Cursor / Replit Agent**

---

Создай React SPA — прототип мобильного приложения JOBS. Это сервис онлайн-рекрутинга через короткие вертикальные видео: соискатели записывают видео-профили, компании публикуют видео-вакансии, отбор происходит через ленту как в TikTok.

---

## ДИЗАЙН

Тёмная тема. Цвета строго:
- Фон: `#0A0A0A`
- Поверхности/карточки: `#141414`
- Акцент: `#BFFF00` (кислотный лаймовый) — кнопки CTA, активные элементы, выделения
- Текст: `#FFFFFF` основной, `#888888` вторичный
- Бордеры: `#1E1E1E`

Шрифты: подключи Google Fonts — `Unbounded` для заголовков (жирный), `Manrope` для интерфейса.

Всё приложение — мобильный формат, max-width 420px, по центру экрана. Нижняя навигация (tab bar). Никаких десктопных лэйаутов.

---

## РОЛИ И РОУТИНГ

Стартовый экран — выбор роли: **Соискатель** или **Компания** (две большие кнопки). Выбор сохраняется в useState. После выбора — разные навигации и экраны.

---

## ЭКРАНЫ СОИСКАТЕЛЯ

### 1. Лента вакансий (главный экран)

Вертикальный полноэкранный scroll-snap список. Каждая карточка — 100vh, видео 9:16.

Вместо реального видео используй цветные градиентные плейсхолдеры (разные для каждой карточки). Поверх плейсхолдера — оверлей:

Снизу:
- Логотип компании (цветной круг с буквой) + название компании
- Название вакансии (крупно, Unbounded)
- Локация · Формат работы
- Теги навыков (#продажи #b2b)

Справа (вертикальная панель иконок как в TikTok):
- ♥ Нравится (счётчик)
- ↩ Откликнуться → переход на экран записи
- ★ Сохранить

Вверху — горизонтальные chips-фильтры: Все / Продажи / HR / Маркетинг / Сервис / Ритейл

Mock-данные: 5 вакансий с разными компаниями, должностями, тегами.

### 2. Запись отклика

Полноэкранный тёмный экран. Сверху — название вакансии и компании.

Блок вопросов от компании (2 вопроса, mock текст). 

По центру — большой круг (кнопка записи). До нажатия: белый с иконкой камеры. После нажатия: красный, вокруг него анимированное кольцо-прогресс (60 секунд). Таймер обратного отсчёта внутри.

После "записи" (через 3 сек имитация): показать превью-плейсхолдер + две кнопки: «Переснять» (outline) и «Отправить» (акцентная, `#BFFF00`, текст чёрный).

### 3. Мои отклики

Список карточек. Каждая: логотип компании + вакансия + дата + статус-бейдж.
Статусы: «Отправлен» (серый), «Просмотрен» (жёлтый), «Ответили» (зелёный `#BFFF00`).

### 4. Профиль соискателя

Аватар (большой круг с инициалами, акцентный цвет) + имя + специальность.
Видео-профиль: прямоугольный плейсхолдер 9:16 с кнопкой play и кнопкой «Перезаписать».
Теги навыков (редактируемые — клик добавляет/удаляет).
Статистика в строку: Просмотры · Отклики · Приглашения.

**Нижняя навигация соискателя:** Лента · Отклики · Профиль

---

## ЭКРАНЫ КОМПАНИИ

### 5. Лента откликов (главный экран компании)

Та же механика scroll-snap, но карточки — это соискатели.

Градиентный плейсхолдер вместо видео. Оверлей снизу: имя кандидата, специальность, краткое описание.

Правая панель:
- ✓ Подходит — анимация: карточка улетает вправо
- ✗ Не подходит — карточка улетает влево  
- ✉ Написать — переход в контакты

Вкладки вверху: «Отклики на вакансии» / «Все кандидаты»

Mock-данные: 5 кандидатов с именами, специальностями, навыками.

### 6. Создать вакансию

Двухшаговая форма.

Шаг 1: поля — Должность (input), Сфера (select), Формат (pills: Офис/Удалёнка/Гибрид), Зарплата от/до (два input), Вопросы к кандидату (до 3 полей, кнопка «+ Добавить вопрос»). Кнопка «Далее →».

Шаг 2: большой прямоугольник-плейсхолдер «Загрузить видео о вакансии» с иконкой + надписью. Кнопка «Опубликовать» — при нажатии показать модалку-успех с анимацией (большая галочка в круге `#BFFF00`, текст «Вакансия опубликована!», кнопка «К откликам»).

### 7. Контакты

Список кандидатов из шортлиста (тех, кому нажали ✓). Карточка: аватар + имя + вакансия + кнопка «Написать». Кнопка открывает простой мини-чат (UI без реальной отправки): поле ввода + кнопка отправки, история сообщений mock.

### 8. Профиль компании

Логотип (цветной круг с буквой) + название + отрасль.
Видео о компании: плейсхолдер 9:16 с кнопкой «Обновить видео».
Короткое описание компании (текст).
Список активных вакансий (компактные карточки: должность + просмотры + отклики).

**Нижняя навигация компании:** Отклики · + Вакансия · Контакты · Профиль

---

## АНИМАЦИИ

- Scroll-snap в лентах: плавный momentum-скролл между карточками
- Кнопка «Откликнуться»: пульсирующий glow эффект цветом `#BFFF00`
- Карточки в ленте компании: framer-motion drag по горизонтали, при dragEnd > 100px — улетает в сторону + opacity 0
- Таб-бар: активный таб подсвечивается `#BFFF00`, плавный transition
- Переходы между экранами: fade или slide (200ms)
- Форма вакансии: прогресс-бар шагов сверху

## ВАЖНО

- Приложение полностью работает без backend — только useState / useReducer
- Не использовать localStorage
- Все переходы через React Router или простой useState-роутер
- Код в одном файле App.jsx или разбить на компоненты — на усмотрение
- Никакого Tailwind — только CSS-in-JS или styled-components или CSS modules
- Сделать красиво: это pitch-прототип для инвесторов
